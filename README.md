# python-projects

**Guess the number game.py** — A fun Python guessing game where the user tries to guess a randomly generated number with hints after each attempt.
**Febonnoci series.py** — Generates the Fibonacci sequence using loops and basic Python logic.
**Level 1 - python projects.py** — A collection of beginner-friendly Python mini projects created while learning core programming concepts.
**Number of letters within the name.py** — Counts and displays the number of letters in a user’s name.
**Number_List.py** — Analyzes a list of numbers by calculating the sum, average, minimum, and maximum values.
**Palindrome.py** — Checks whether a word or phrase is a palindrome by comparing characters forwards and backwards.
**area of the rectangle** — Calculates the area of a rectangle using user-provided length and width values.
**calculator.py** — A simple calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division.
**temperature_converter.py** — Converts temperatures between Celsius and Fahrenheit with user input.
